# 可研报告编写规范

## 语言与格式

### 语言风格
- 客观、严谨、规范的工程咨询语言
- 使用第三人称，避免主观表述
- 术语统一，同一概念全文用同一术语
- 不使用口语化表达、网络用语

### 数字格式
- 财务数据保留两位小数（金额以万元为单位）
- 面积保留整数（平方米）
- 百分比保留两位小数
- 大数字用"万元""亿元"为单位

### 表格规范
- 所有表格必须有编号和标题
- 金额列居中，文字列居中
- 总计行加粗

### 字数深度标准

| 章节 | 下限 | 上限 | 不达标时补充方向 |
|------|:---:|:---:|------|
| 第1章 概述 | 3,000 | 5,000 | — |
| 第2章 建设背景和必要性 | 10,000 | 20,000 | 增加政策逐条对照、区域数据对比、案例分析 |
| 第3章 需求分析与产出 | 15,000 | 30,000 | 展开预测模型推导、多情景分析、细化产出规格 |
| 第4章 选址与要素保障 | 10,000 | 20,000 | 增加选址比选方案、要素量化分析 |
| 第5章 建设方案 | 30,000 | 50,000 | 六大板块每块展开到5k~8k字 |
| 第6章 运营方案 | 10,000 | 15,000 | 细化组织架构、岗位编制、绩效KPI |
| 第7章 投融资与财务 | 15,000 | 30,000 | 介绍收入、收入可实现论证、分析成本、计算财务指标、展开财务模型推导、多情景敏感性分析 |
| 第8章 影响效果分析 | 15,000 | 30,000 | 逐项量化经济/社会/生态影响 |
| 第9章 风险管控 | 1,0000 | 20,000 | 逐风险维度展开识别-评估-应对 |
| 第10章 结论与建议 | 3,000 | 5,000 | — |

> 以上字数为正文文字量，不含表格数据。复杂项目取上限，简单项目取下限，浮动 ±30%。

## 质量检查清单

### 每章完成后检查
- [ ] 该章数据是否有来源标注？（≥3个独立来源）
- [ ] 估算类数据是否说明估算依据？
- [ ] 是否有无法确认的内容以【待确认】标注？
- [ ] 逻辑是否闭环？（结论有论据支撑）
- [ ] 术语是否统一？
- [ ] 字数是否达标？（见下方字数标准）

### 全报告完成后检查

#### 数据一致性
- [ ] 各章中的项目名称完全一致
- [ ] 第7章投资估算总数与第1章概述一致
- [ ] 第7章投资与第5章技术方案的建设内容匹配
- [ ] 第3章建设规模在各章中统一
- [ ] 第5章工期与第1章概述一致

#### ⚠️ 第5章-第7章依赖校验（逻辑闭环关键项）

> **核心原则**：第5章（建设方案）是第7章（投资估算）的技术依据。投资估算中的每一笔建筑工程费、安装工程费、设备购置费都应对应第5章中确定的技术参数。

| 检查项 | 第7章数据 | 第5章对应依据 | 是否一致 |
|--------|----------|--------------|---------|
| 建筑面积合计 | _____ ㎡ | 第5章各建筑单体面积之和 = _____ ㎡ | |
| 建筑结构形式 | _____ | 第5章技术方案确定的结构形式 | |
| 建筑工程单价 | _____ 元/㎡ | 第5章确定的建造标准是否支撑该单价 | |
| 设备清单及数量 | 共计 ___ 台/套 | 第5章设备方案中的清单 | |
| 室外工程面积 | _____ ㎡ | 第5章总平面布置中的工程量，如道路、绿化、管线管网等 | |
| 建设工期 | _____ 月 | 第5章建设管理方案中的工期是否符合工期定额 | |

**校验规则**：
- 如第7章先于第5章编写，则第7章投资估算标注为 **"初步匡算"**，单价依据标注为 **"经验指标"**
- 第5章完成后，必须回核第7章，逐项对照上表
- 不一致时以第5章技术方案为依据，修正第7章估算
- 修正后在第5章末尾增加 **"5.X 投资估算复核"** 节，记录复核结果和修正内容

#### 逻辑完整性
- [ ] 第2章必要性论证有需求数据支撑
- [ ] 第3章建设规模有需求分析支撑
- [ ] 第5章技术方案与第7章投资估算匹配
- [ ] 第7章资金来源与投资额匹配
- [ ] 第9章风险分析覆盖全生命周期七大维度

#### 七大维度覆盖检查（发改委2023大纲要求）
- [ ] 建设必要性 — 第2章论证充分
- [ ] 要素保障性 — 第4章论证充分
- [ ] 工程可行性 — 第5章论证充分
- [ ] 运营有效性 — 第6章论证充分
- [ ] 财务合理性 — 第7章论证充分
- [ ] 影响可持续性 — 第8章论证充分
- [ ] 风险可控性 — 第9章论证充分

#### 遗漏项检查
- [ ] 是否有大纲要求的章节未编写？
- [ ] 第7章投资估算是否漏项？（土地费、预备费、建设期利息等）
- [ ] 第8章是否包含碳达峰碳中和分析？（如适用）
- [ ] 第5章是否包含数字化方案？（如适用）
- [ ] 第6章是否包含绩效管理方案？

#### 合规性
- [ ] 编制依据是否包含最新有效法规？
- [ ] 是否满足发改投资规〔2023〕304号深度要求？
- [ ] 引用标准是否现行有效？

#### 文字质量
- [ ] 术语是否准确？
- [ ] 表述是否清晰无歧义？
- [ ] 是否有错别字或标点错误？
- [ ] 章节编号是否连续？

## 联网研究的来源优先级

1. **政府官网**（.gov.cn）— 政策法规、统计数据
2. **发改委/住建部门公告** — 规划文件、标准发布
3. **行业协会/学会** — 技术标准、行业数据
4. **权威数据库** — 统计年鉴、市场研究报告
5. **地方政府部门信息** — 估算指标、概算指标
6. **同类项目公示** — 环评公示、招投标公告中的项目信息
