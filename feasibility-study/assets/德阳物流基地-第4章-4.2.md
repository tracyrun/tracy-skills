## 4.2 项目建设条件

### 4.2.1 自然环境条件

**（1）地形地貌**

项目选址区域位于成都平原东北边缘与龙泉山脉低山丘陵过渡地带，总体地势东北高、西南低，属浅丘地貌。地块内海拔高程在465~485m之间，相对高差约20m，整体坡度小于5%，地势较为平缓开阔，适宜大型物流仓库和货物堆场的大规模平面布局。地块内无滑坡、崩塌、泥石流等不良地质现象。

**（2）气象条件**

项目区属亚热带湿润季风气候，四季分明，气候温和，无霜期长，雨量充沛。主要气象参数如下：

| 气象要素 | 数值 | 备注 |
|---------|:---:|------|
| 多年平均气温 | 16.1~17.0℃ | — |
| 极端最高气温 | 36.5℃ | — |
| 极端最低气温 | -6.7℃ | — |
| 年平均降水量 | 825~893mm | — |
| 一日最大降水量 | 222.8mm | — |
| 年平均雨日 | 137天 | — |
| 夏半年（5~10月）降水量占比 | 89% | 降水高度集中 |
| 平均相对湿度 | 81% | — |
| 平均风速 | 1.4~1.6 m/s | — |
| 主导风向 | 东北风（NNE） | — |
| 最大风速 | 17.3 m/s | — |
| 无霜期 | 271~290天 | — |

气象条件对项目的影响评价：项目区夏季降水集中，需重点考虑场地排水和防洪设计；风速较小，对钢结构建筑的抗风设计压力适中；无霜期长，冬季施工基本不受低温影响；极端最高气温不高，冷库制冷负荷处于合理区间。

**（3）水文条件**

项目区属沱江流域绵远河水系。绵远河为沱江上游干流，自北向南从项目地块东侧约2.5公里处流过，境内流长约17.3公里。主要支流东河（绵远河左岸一级支流，总河长9.63km，集雨面积22.3km²）位于地块东北侧约4公里处。项目地块不在河道管理范围内，不受百年一遇洪水威胁。

区域灌溉渠系包括人民渠33、34、43支渠及人民渠五期、七期干渠，其中最近的人民渠支渠位于地块西侧约500m处，不穿越项目地块，不影响项目建设和运营。

地下水分上下两层：上层埋深8~20m，为第四系松散层孔隙潜水，富水性一般；下层埋深20~40m，为基岩裂隙水，水量较小。地下水以重碳酸钙-镁型水为主，水质达国家二级标准，对混凝土和钢结构无腐蚀性。

**（4）工程地质条件**

根据区域地质资料和邻近项目勘察成果（参考德阳蓉欧冷链运营中心、德阳临港制造产业园地质勘察报告），项目区地质条件概述如下：

| 地质要素 | 特征描述 |
|---------|---------|
| 区域构造 | 位于四川盆地川西台陷北部，区域构造稳定性较好 |
| 地震动峰值加速度 | 0.10g（对应地震基本烈度Ⅶ度） |
| 地震动反应谱特征周期 | 0.35s |
| 地表覆盖层 | 耕植土（厚0.3~0.5m）+ 第四系残坡积粉质粘土（厚2~8m） |
| 下卧基岩 | 侏罗系蓬莱镇组砂岩、泥岩互层 |
| 地基承载力特征值 | 粉质粘土 fak=150~200kPa，基岩 fak=500~800kPa |
| 场地土类型 | 中硬土，Ⅱ类建筑场地 |
| 不良地质作用 | 不发育 |
| 地下水腐蚀性 | 对混凝土和钢结构无腐蚀性 |

**（5）地震条件**

项目区地震动峰值加速度为0.10g，地震基本烈度为Ⅶ度，设计地震分组为第二组。依据《建筑抗震设计规范》（GB 50011-2010，2016年修订版），本项目抗震设防烈度按Ⅶ度执行，设计基本地震加速度值取0.10g。场地类别为Ⅱ类，属抗震一般地段，适宜建设。

**（6）防洪条件**

项目地块位于绵远河左岸约2.5km处，地势高于绵远河百年一遇洪水位约15m以上，不受绵远河洪水淹没威胁。地块内部排水通过规划雨水管网接入物流港片区市政雨水系统，最终排入绵远河。区内无历史严重内涝记录（2000年特大暴雨造成黄许镇镇区局部内涝，本项目地块位于镇区以北开阔地带，地势较高，未受影响）。

> **来源**：黄许镇百度百科自然地理数据；区域地质资料参考德阳蓉欧冷链运营中心、德阳临港制造产业园地质勘察报告；《建筑抗震设计规范》（GB 50011-2010，2016年修订版）；《中国地震动参数区划图》（GB 18306-2015）

### 4.2.2 交通运输条件

**（1）铁路运输条件**

项目地块东距宝成铁路黄许货运站约3公里，公路运输约10分钟可达。黄许站是宝成线入川路段最大的货运站，现有到发线6条、货物线4条，年货运吞吐能力超1,000万吨，主要办理大宗物资（钢材、粮食、化肥、建材等）的整车到发业务。黄许站已开通中欧班列、中老班列等国际班列业务，截至2025年底德阳国际铁路物流港累计开行国际班列454列，联通30余个国家、60余座城市。

此外，德阳境内还有德阳站（宝成线、成绵乐客专）、德阳南站（宝成线）等铁路站场，距离项目地块约12~15公里，可提供补充铁路运输能力。

**（2）公路运输条件**

项目周边公路运输条件优越：

- **高速公路**：南距成都第三绕城高速（G4203）孝感互通约5公里，西距京昆高速（G5）德阳北互通约8公里，南距成绵高速复线（S1）约15公里。通过高速路网可在1小时内到达成都、绵阳，2小时内到达广元、遂宁、南充等城市。
- **国省干线**：G108国道从地块东侧约1公里处南北向通过，是连接德阳与绵阳、广元的主要干线公路。
- **城市快速路**：德罗大道（德阳—罗江快速通道）已于2026年建成通车，地块向东约3公里可达，成为连接德阳城区与罗江区的南北向交通主轴。一环路（德阳市区环线）从地块南侧约6公里处通过。
- **物流港内部路网**：德阳国际铁路物流港规划"四横六纵"主干路网，其中银山路（南北向，紧邻地块西侧）和涟江路（东西向，紧邻地块南侧）为物流港最主要的货运通道。银山路南延线（钱塘江路—涟江路段）已纳入《德阳市2025年度100个重大项目》，前期工作推进中。

**（3）航空运输条件**

项目地块距绵阳南郊机场约55公里（车程约50分钟），距成都双流国际机场约80公里（车程约70分钟），距成都天府国际机场约100公里（车程约80分钟），三个机场均在1.5小时车程圈内，可满足高端冷链和快递物流的航空转运需求。其中成都双流国际机场和天府国际机场均为4F级国际枢纽机场，已开通至全球主要城市的货运航线。

**（4）多式联运条件综合评价**

本项目具备公铁联运的天然优势：大宗物资（钢材、粮食、建材等）可通过宝成铁路黄许站大批量低成本到达，经3公里短驳公路运输进入本项目仓储中转，再通过公路分拨配送到德阳及周边地区的终端客户。德阳国际铁路物流港海关监管中心（2025年12月启用）可为出口货物提供"本地通关、铁路出境"的便利条件，实现公铁海多式联运。

> **来源**：德阳国际铁路物流港控制性详细规划（2025年11月批后公布）；德阳市交通运输局德罗大道通车公告（2026年）；德阳国际铁路物流港海关监管中心启用新闻报道（2025年12月31日，德阳市人民政府官网）

### 4.2.3 公用工程条件

**（1）供水条件**

旌阳区正在实施全域供水建设项目，总投资约13.97亿元，覆盖黄许镇等10个乡镇/街道，预计2025年底全部竣工。建设内容包括新建净水厂2座、中途加压泵站5座、取水口2座、改造供水站33座及配套供水管网。

项目地块供水拟从物流港片区市政供水管网DN315~DN400主管接入，接口位于地块西侧银山路或南侧涟江路，水压≥0.25MPa，可满足项目日用水量约300m³/d的需求（含生活用水、绿化灌溉用水、消防补水等）。消防用水由项目自建消防水池（有效容积≥800m³）+ 消防泵房保障，市政供水作为补水水源。

黄许镇南华宫片区配套给水工程已于2025年9月完成施工招标，新建DN315PE管239m、DN250PE管344m及阀井5座，进一步提升了区域供水可靠性。

**（2）排水条件**

项目采取雨污分流制。雨水经园区雨水管网收集后，通过海绵设施（下凹式绿地、雨水花园、雨水调蓄池）滞蓄净化，超标雨水排入物流港片区市政雨水管网，最终排入绵远河。污水主要为生活污水（约50m³/d），经化粪池预处理后排入物流港片区市政污水管网，送至黄许镇污水处理站集中处理。

旌阳区全域供水建设项目中包含改扩建污水处理厂（站）3座、新建污水处理站16座及配套污水管网，区域污水处理能力将得到显著提升。物流港片区污水处理厂已纳入"十五五"基础设施配套规划。

**（3）供电条件**

项目地块位于德阳电网覆盖范围内，区域有220kV德阳变电站（距项目约8公里）和110kV黄许变电站（距项目约3公里）。德阳是四川省重要电力输出市，电力供应充沛，近年来未发生工业限电情况。

项目供电拟从物流港片区10kV市政电力线路双回路接入，经项目自建10kV开闭所分配至各变配电房。项目总装机容量约7,200kVA，计算负荷约5,100kW，拟配置2台2,000kVA + 2台1,600kVA干式变压器，另设1台500kW柴油发电机组作为应急备用电源。物流港片区"十五五"电力配套规划已纳入德阳供电公司的配电网建设计划。

**（4）供气条件**

项目地块位于德阳市华润燃气有限公司供气范围，物流港片区已铺设城镇燃气管网。项目用气需求较小（主要为辅助用房食堂、供暖用气），日高峰用气量约200Nm³/d，可从银山路或涟江路市政中压燃气管网（0.4MPa）接入，经调压箱调至低压后使用。

**（5）通信条件**

项目地块位于德阳市通信网络全覆盖范围内，中国电信、中国移动、中国联通、中国广电四大运营商光纤网络已覆盖至物流港片区。项目可从涟江路或银山路市政通信管沟引入多芯光纤，满足园区办公网络、监控网络、WMS/TMS/OMS系统运行、工业互联网等高带宽需求。4G/5G信号全覆盖，可支持园区无线终端、RFID、PDA等移动设备的实时通信。

**（6）消防条件**

项目按一级耐火等级设计，自建消防系统（消防水池800m³+消防泵房+室内外消火栓+ESFR自动喷水灭火系统+火灾自动报警系统）。物流港片区规划有特勤消防站1座，距项目地块约5公里，出警到达时间约10分钟，可作为项目消防补充保障。

> **来源**：旌阳区供水建设项目进展报道（2025年8月，德阳文明网）；黄许镇南华宫片区配套给水工程招标公告（2025年9月）；德阳国际铁路物流港控制性详细规划（2025年11月）；德阳国际铁路物流港开发建设中心关于"十五五"水电气基础设施配套规划（2026年）

### 4.2.4 施工条件

**（1）场地条件**

项目地块地势较为平坦开阔，高差约20m，场地平整工程量可控（预估挖填方量约40万m³）。地块内无大型建筑物和构造物，仅少量农田和零星农房，拆迁量小。场地周边无军事设施、文物古迹、自然保护区等特殊限制条件。

**（2）施工交通条件**

地块西临银山路（在建），南临涟江路（已建成银山路口段），东侧和北侧为规划道路。施工期间可利用涟江路和现有村道作为施工进场道路，开辟不少于2个施工出入口。大宗建筑材料（钢材、商品混凝土）可从德阳市区经G108国道和德罗大道运输至现场，运距约10~15公里。

**（3）施工用水用电条件**

施工用水可从物流港片区市政供水管网临时接入，设置临时水表和施工用水管路。施工用电可从银山路10kV电力线路临时接入，设置1~2台临时变压器（400~630kVA），满足打桩、焊接、塔吊等施工机械用电需求。施工期间还需自备柴油发电机作为应急备用电源。

**（4）建筑材料供应条件**

德阳市是四川省重要的建筑钢材和水泥生产基地，区域内主要建筑材料供应充足：
- **钢材**：德阳有攀钢集团、川威集团等钢材生产基地和多个大型钢材市场，Q355B结构钢材可就地采购，运距约15~30公里。
- **商品混凝土**：黄许镇及周边有多家商品混凝土搅拌站（C30~C50均可供应），运距约5~10公里。
- **砂石料**：绵远河沿线有多个砂石料场，砂石料可就近采购。
- **钢构件加工**：德阳市区及广汉市有多家钢结构加工企业，门式刚架和钢梁钢柱可在工厂预制后运至现场吊装，运距约15~30公里。

**（5）施工气象条件**

项目区年平均无霜期271~290天，冬季气温很少低于0℃，全年有效施工天数约280~300天。降水集中在5~10月（占全年89%），夏季暴雨可能造成短期停工，但总体可施工窗口期较长。主要制约因素是夏季高温（极端最高36.5℃）和暴雨天气，需在施工组织设计中合理安排雨季施工措施和防暑降温方案。

**（6）施工生活条件**

黄许镇镇区距项目地块约2公里，有较为完善的生活配套设施（住宿、餐饮、医疗、教育等），可为施工人员提供基本生活保障。德阳市区距项目约8公里，可提供更全面的施工管理和技术服务支撑。
