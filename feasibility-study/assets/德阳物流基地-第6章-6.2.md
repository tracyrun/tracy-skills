## 6.2 运营组织方案

### 6.2.1 组织机构设置

运营管理公司按照"精简高效、权责分明、专业分工"的原则设置组织机构。公司设总经理1名、副总经理2名，下设5个职能部门和1个监控调度中心。

**组织架构图**：

```
                         ┌─────────────┐
                         │   董事会     │
                         └──────┬──────┘
                                │
                         ┌──────┴──────┐
                         │   监事会     │
                         └──────┬──────┘
                                │
                         ┌──────┴──────┐
                         │   总经理     │
                         └──────┬──────┘
                                │
              ┌─────────────────┼─────────────────┐
              │                 │                 │
     ┌───────┴───────┐ ┌───────┴───────┐ ┌───────┴───────┐
     │ 副总经理(运营) │ │ 副总经理(管理) │ │   监控调度中心  │
     └───────┬───────┘ └───────┬───────┘ └───────────────┘
              │                 │
    ┌─────────┼─────────┐      ┌─────────┼─────────┐
    │         │         │      │         │         │
┌───┴──┐ ┌───┴──┐ ┌───┴──┐ ┌───┴──┐ ┌───┴──┐ ┌───┴──┐
│仓储  │ │运输  │ │客户  │ │综合  │ │财务  │ │安全  │
│运营部│ │调度部│ │服务部│ │管理部│ │资产部│ │环保部│
└──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘
```

**各部门职责与编制**：

| 部门 | 职责 | 编制（人） | 备注 |
|------|------|:---:|------|
| **仓储运营部** | 仓库日常运营管理、库位分配与调度、库存管理与盘点、仓储作业（装卸搬运拣选）外包管理、货架和设备使用管理、冷链温区管理 | 12~15 | 下设大宗仓储组（7人）、冷链组（3人）、中转组（2人） |
| **运输调度部** | 入园/出园车辆调度、运输计划编制、自有及外包车辆管理、运输路线优化、多式联运协调（铁路短驳衔接）、司机管理 | 5~7 | — |
| **客户服务部** | 客户开发与关系维护、合同谈判与签订、客户订单受理与跟踪、客户投诉处理、收费核算与账单管理、增值服务策划 | 5~6 | 兼市场开发职能 |
| **综合管理部** | 人事行政管理、档案管理、法律事务与合同管理、后勤保障、对外联络与接待、企业文化建设 | 4~5 | — |
| **财务资产部** | 财务核算与报表编制、预算管理、成本控制、资金管理、资产台账管理、税务管理 | 3~4 | — |
| **安全环保部** | 安全生产监督管理、消防安全管理、职业健康管理、环境保护管理、应急预案编制与演练、事故调查处理、安全培训 | 4~5 | 含氨制冷专项安全员1人 |
| **监控调度中心** | 园区视频监控7×24值班、消防报警监控、冷库温湿度监控、能耗监控、出入口车牌识别管理、突发事件调度指挥 | 8~10 | 四班三运转（每班2人+值班主管） |
| **经营管理层** | 总经理1人、副总经理2人 | 3 | — |
| **合计** | | **44~55** | |

> 注：上述编制为运营管理公司自有人员。仓储作业（装卸、搬运、拣选等）和保安、保洁等外包人员约100~120人，由外包服务商配置和管理，不计入运营管理公司编制。

### 6.2.3 员工培训计划

**（1）岗前培训**（上岗前，2~4周）

全体员工须完成以下内容的培训并考核合格后方可上岗：

- 公司管理制度和岗位职责（1天）
- 园区功能布局和物流动线（1天）
- WMS/TMS/OMS/WCS系统操作（3~5天，操作类岗位）
- 安全生产和消防知识（2天，含灭火器实际操作）
- 冷链作业规范和氨制冷安全（1天，冷链相关人员）
- 叉车安全操作规程（1天，涉及叉车监管人员）
- 客户服务礼仪和投诉处理（1天）
- 应急预案和紧急疏散演练（1天）

**（2）在岗培训**（持续进行）

| 培训类型 | 频次 | 对象 | 内容 |
|---------|:---:|------|------|
| 月度安全培训 | 每月1次 | 全体员工 | 当月安全隐患通报、典型事故案例分析、季节性安全注意事项 |
| 专业技能提升 | 每季1次 | 按岗位分批 | 仓储新技术、信息化系统更新、冷链运维技术等 |
| 管理层培训 | 每半年1次 | 部门经理及以上 | 现代物流管理、供应链优化、领导力提升 |
| 应急演练 | 每半年1次 | 全体员工 | 火灾、氨泄漏、暴雨内涝、大面积停电等情景实战演练 |
| 外部参观学习 | 每年1~2次 | 骨干员工 | 赴传化、普洛斯、蓉欧冷链等先进园区交流学习 |

**（3）特种作业人员培训**

- 叉车司机（外包人员）：须持有《特种设备作业人员证》（N1），每4年复审，由外包商确保资质有效性
- 电工作业人员：须持有《特种作业操作证》（电工作业），每3年复审
- 消防设施操作员：监控中心值班人员须持中级以上《消防设施操作员证》
- 氨制冷操作人员（外包人员）：须经过氨制冷专项安全培训并考核合格

> **来源**：《特种设备作业人员考核规则》（TSG Z6001）；《安全生产培训管理办法》（国家安全生产监督管理总局令第44号）
